Security Overview

Duoying (Shenzhen) Supply Chain Co., Ltd. maintains reasonable information security measures designed to protect company systems, business data, and relevant personal information processed through our ERP system and related services.

Our Security Measures Include

  • Access limited to authorized personnel only
  • Least-privilege access control
  • Secure password practices
  • MFA for important administrative accounts
  • HTTPS/TLS encryption for data transmission where applicable
  • Encryption at rest within applicable AWS storage services
  • Timely application of security updates and patches
  • Logging and monitoring for abnormal activity
  • Prompt reporting and handling of suspected security incidents

For security or privacy-related questions, please contact: contact@dyeos.net
